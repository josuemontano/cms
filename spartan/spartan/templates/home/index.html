ATC Talentos Creativos le presenta SpartanCMS 4.0, un gestor de contenidos para su sitio web hecho a medida.

SpartanCMS le entrega el control sobre su sitio, de tal forma que administrarlo sea tan sencillo como editar un documento de texto. El menú (hallado a la izquierda de esta página) le permite navegar por las distintas secciones de SpartanCMS. A continuación se describen las secciones principales de este administrador de contenidos.

Páginas

Esta sección le permitirá acceder y editar el contenido de su sitio. Puede cambiar el orden de las páginas, quitarlas, ocultarlas o añadir nuevas.

Sitio

Aquí es donde puede administrar algunas propiedades globales, como ser el nombre del sitio, el logo y sus correos de contacto.

No olvide escribir una descripcion y las palabras clave de su sitio, esto hace posible mejores resultados cuando su sitio se busca mediante un motor de búsqueda como Google.

Usuarios

La administración de usuarios se puede realizar desde esta sección. Puede cambiar contraseñas y grupos de los usuarios existentes, además de crear nuevos.


Si tiene alguna consulta, sugerencia o problema puede escribirnos a info@talentoscreativos.com, estamos para ayudarle.
